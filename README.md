# Vuelvo Cards — Landing Page 

Sitio web oficial de [Vuelvo Cards](https://vuelvocards.com) — plataforma de lealtad digital para negocios en El Salvador y LATAM.

## Descripción

Landing page de una sola página (`index.html`) construida en HTML/CSS/JS puro. Sin frameworks, sin dependencias externas excepto Google Fonts.

## Estructura

```
index.html       # Página completa — todo el código vive aquí
README.md        # Este archivo
```

## Secciones

1. **Nav** — Navegación fija con links de anclaje
2. **Hero** — Headline dinámico + mockup de wallet
3. **Logo Strip** — Marquee con negocios clientes
4. **Problema** — 4 cards con hover que muestra la solución
5. **Cómo funciona** — Layout imagen + pasos estilo Perkstar
6. **Tarjetas** — Carrusel con los 8 tipos de programa
7. **Personalización** — Widget interactivo de preview en tiempo real
8. **Herramientas** — Tabs con animaciones por feature
9. **Precios** — Toggle mensual/anual con 3 planes
10. **FAQ** — Accordion con 12 preguntas
11. **CTA** — Llamada a acción final
12. **Footer** — Links, redes y contacto

## Imágenes

Las imágenes están referenciadas desde WordPress Media Library:
- `https://vuelvocards.com/wp-content/uploads/...`

Para actualizar una imagen, súbela a WordPress y cambia la URL en `index.html`.

## Deployment

El sitio está deployado en **Vercel** conectado a este repositorio.

Cada push a `main` redespliega automáticamente.
